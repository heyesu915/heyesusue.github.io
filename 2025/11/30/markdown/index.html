Markdown 语法入门教程（新手友好版）

Markdown 是一种轻量级标记语言，由约翰·格鲁伯（John Gruber）于2004年设计。它用简单的符号标记文本格式，无需复杂的排版操作，写出来的内容既易读易写，又能快速转换为 HTML、PDF 等格式，广泛用于笔记、文档、博客、GitHub 说明文档（README.md）等场景。

本文从新手视角，讲解 Markdown 核心语法，每个语法点都配示例，看完就能上手！

一、基础核心语法（必学）

1. 标题

Markdown 用 # 表示标题，# 数量对应标题级别（1-6级），# 和标题文字之间必须加空格。

语法 效果

一级标题
## 二级标题

### 三级标题

#### 四级标题

##### 五级标题
五级标题
###### 六级标题
六级标题

新手建议：日常使用1-3级标题即可，层级过多会导致文档结构混乱。

  • 段落：直接输入文字即可，段落之间用空行分隔（按两次 Enter）；
  • 换行：在一行文字末尾加两个及以上空格，再按 Enter（或直接加 <br> 标签）。

三、新手常见注意事项

  1. 空格的重要性：标题、列表、引用等语法中，符号和文字之间必须加空格（如 # 标题 正确，#标题 错误）；
  2. 编辑器兼容：不同编辑器（如 Typora、GitHub、Notion）对 Markdown 语法的支持略有差异（比如部分平台不支持脚注），以预览效果为准；
  3. 简化学习：先掌握「标题、列表、强调、链接、代码块」核心语法，进阶语法按需学习；
  4. 实时预览：写 Markdown 时开启预览模式，及时检查格式是否正确。

四、新手推荐工具

1. 在线编辑器（无需安装）

  • Markdown 编辑器（菜鸟工具）：新手友好，实时预览；
  • StackEdit：功能全，支持同步到云盘；
  • GitHub Gist：适合写代码片段。

2. 客户端工具（本地使用）

  • Typora：所见即所得，新手首选（免费版可用）；
  • VS Code + 插件（Markdown All in One）：代码+文档一体化；
  • Obsidian：适合写笔记、知识库，全平台同步。
